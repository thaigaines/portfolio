Projects

AI vs S&P 500 Analysis

Analyzed several high-profile AI-related stocks to track their performance relative to the broader U.S. market (S&P 500). Utilized:
  • Price normalization to enable direct comparison across stocks of different scales
  • Month-to-month percentage changes to observe growth patterns and volatility
The analysis aimed to identify signs of potential bubble behavior, pointing out stocks with rapid growth and high volatility relative to the market.

Stock Explorer Tutorial

An interactive tutorial in Python (Jupyter Notebook) demonstrating financial data analysis and visualization.
  • Pulled stock market data using the yfinance API
  • Created visualizations with matplotlib to explore price trends and performance
  • Designed as both a practical exercise in financial analytics and a beginner-friendly guide to using Python tools for market exploration
